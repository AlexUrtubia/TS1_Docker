# Proyecto Docker - Entorno Web Completo

## Descripción
Este proyecto implementa un entorno web utilizando Docker, compuesto por:

- Frontend: React servido con Nginx
- Backend: API Node.js (Express)
- Base de datos: PostgreSQL con persistencia

## Arquitectura

Frontend (Nginx)
↓
Backend (Express)
↓
PostgreSQL

## Tecnologías utilizadas

- Docker
- Docker Compose
- Node.js
- Express
- React
- PostgreSQL
- Nginx

## Requisitos

- Docker Desktop instalado
- Docker Compose
